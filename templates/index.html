Quick Start Guide

Step 1: Register for an API Key

Sign up for a free account to get your API key:

POST /api/auth/register
{
  "username": "your_username",
  "email": "your@email.com",
  "password": "your_password"
}

Step 2: Get Your Access Token

Use your credentials to obtain a JWT token:

POST /api/auth/token
{
  "username": "your_username",
  "password": "your_password"
}

Step 3: Make API Requests

Use your token to make authenticated requests:

GET /api/timesync/convert?utc_timestamp=2023-10-15T12:00:00Z&target_timezone=America/New_York
Authorization: Bearer your_jwt_token

Step 4: Parse the Response

Process the JSON response:

{
  "utc_timestamp": "2023-10-15T12:00:00+00:00",
  "local_timestamp": "2023-10-15T08:00:00-04:00",
  "timezone": "America/New_York",
  "offset": "-04:00",
  "is_dst": true
}
